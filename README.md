# 🌌 VibeIQ

> Find places based on how you feel — not just where you go.

VibeIQ is an AI-powered web application that helps users discover cafes, restaurants, and local spots based on their mood, vibe, and emotional preferences.

Instead of searching “cafes near me”, you can search:
- “cozy place to read”
- “late night chill vibe”
- “quiet place to think”

---

## ✨ Features

### 🧠 AI-Powered Vibe Search
Describe your mood in natural language and get personalized place recommendations.

### 🎯 Vibe Match Score
Each result includes a match score showing how well it fits your vibe.

### 💬 Emotional AI Explanations
Human-like descriptions based on:
- lighting
- noise level
- crowd type
- atmosphere

### ❤️ Like / Dislike System
Refine your taste and improve recommendations over time.

### 🧬 Vibe Profile
Track your aesthetic preferences:
- Cozy
- Quiet
- Energetic
- Aesthetic
- Nature

### 💾 Vault (Saved Places)
Save and organize your favorite spots into categories like:
- Work Mode
- Relax
- Night Vibes

### 🔐 Authentication
- Google Sign-In via Supabase

---

## 🛠 Tech Stack

- **Frontend:** React + Vite + TypeScript
- **Styling:** Tailwind CSS
- **Backend / Auth:** Supabase
- **AI Engine:** Google Gemini API
- **Animations:** Framer Motion
- **Hosting:** Vercel
